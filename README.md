 BlockE2T-MORL
======================================

Blockchain-Enabled Multi-Objective Reinforcement Learning Framework
for Secure Energy- and Time-Efficient Smart Path Planning in Cloud
Environments

Research Overview
----------------
This project implements the BlockE2T-MORL framework proposed for
secure and efficient path planning in cloud-based environments.
The framework combines:

- Multi-Agent Reinforcement Learning
- Actor-Critic Learning
- Blockchain-Inspired Trust Management
- Federated Learning
- Multi-Objective Reward Optimization
- Smart Ambulance Routing Scenario

 

Research Structure
-----------------
Part 1 : Environment and Trust Infrastructure
Part 2 : Cloud Grid Environment
Part 3 : Actor-Critic Reinforcement Learning
Part 4 : Blockchain Validation and Federated Learning
Part 5 : Training and Evaluation
Part 6 : Visualization and Performance Analysis

Requirements
------------
Python 3.10 or higher

Install dependencies:

pip install -r requirements.txt

 
The implementation generates:

- Energy Consumption Analysis
- Latency Analysis
- Trust Evolution Curves
- Reward Convergence Graphs
- Success Rate Evaluation
- Federated Learning Performance
